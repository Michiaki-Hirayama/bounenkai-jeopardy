# 社長室 Jeopardy

社長室の忘年会向けのJeopardy風クイズアプリです。

## 🎮 3つの画面

### 1. Jeopardy クイズ画面（メイン）
- 濃い赤ベースのかっこいいデザイン
- カテゴリー別問題グリッド
- 問題表示・回答確認機能
- 画像/動画の表示対応

### 2. 編集画面
- カテゴリーの追加・削除・順番変更
- 問題の追加・編集・削除
- 画像/動画のアップロード
- データのエクスポート/インポート（バックアップ）

### 3. スコアボードアプリ（独立）
- 別ウィンドウで起動可能
- 他のゲームでも再利用可能
- ローカルストレージで自動保存

## 📁 ファイル構成

```
Bounenkai_Geopardy/
├── index.html          # クイズ画面
├── style.css           # クイズ画面のスタイル
├── script.js           # クイズ画面のロジック
├── admin.html          # 編集画面
├── admin.css           # 編集画面のスタイル
├── admin.js            # 編集画面のロジック
├── db.js               # IndexedDB データベース管理
├── README.md           # このファイル
└── scoreboard/         # スコアボードアプリ（独立）
    ├── index.html
    ├── style.css
    └── script.js
```

## 🚀 使い方

### GitHub Pages で公開する場合

1. このフォルダを GitHub リポジトリにプッシュ
2. Settings → Pages → Source を `main` ブランチに設定
3. 公開URLにアクセス

### ローカルで使う場合

```bash
cd /path/to/Bounenkai_Geopardy
python -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセス

## 📝 問題の編集

1. 編集画面（`admin.html`）にアクセス
2. カテゴリーを追加
3. 問題を追加・編集
4. 必要に応じて画像/動画をアップロード
5. データをエクスポートしてバックアップ

## 💾 データ管理

### データの保存場所
- ブラウザの IndexedDB に保存
- ブラウザごとに独立したデータ

### バックアップ方法
1. 編集画面で「📥 エクスポート」ボタンをクリック
2. JSONファイルがダウンロードされる
3. 安全な場所に保管

### 別のPCに移行する場合
1. 元のPCで「📥 エクスポート」
2. 新しいPCで「📤 インポート」

## 🎯 忘年会での使い方

### 事前準備
1. 自分のPCで問題を作成
2. 画像/動画をアップロード
3. データをエクスポートしてバックアップ

### 当日
1. 会場のPCで GitHub Pages のURLにアクセス
2. 編集画面でバックアップファイルをインポート
3. クイズ画面とスコアボードを別ウィンドウで開く
4. ゲーム開始！

## 🔧 カスタマイズ

### デフォルトチームの変更
`scoreboard/script.js` の初期チームを編集：

```javascript
teams = ["Team A", "Team B", "Team C", "Team D"];
```

### デザインの変更
- `style.css` - クイズ画面のデザイン
- `admin.css` - 編集画面のデザイン
- `scoreboard/style.css` - スコアボードのデザイン

## ⚠️ 注意事項

- データはブラウザごとに保存されます
- ブラウザのデータを削除するとデータも消えます
- 重要なデータは必ずエクスポートしてバックアップしてください
- プライベートモードでは保存されない場合があります
